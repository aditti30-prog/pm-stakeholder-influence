# PM Stakeholder Influence Playbook

A Claude Skill for navigating difficult stakeholder conversations.
9 situation types. 8 exec archetypes. 10 tactical plays with exact scripted language.

## Install (30 seconds)

1. Download `pm-stakeholder-influence.zip` from this repo
2. Claude.ai → Settings → Customize → Skills
3. Click "+" → "Create skill" → Upload the zip
4. Toggle on. Done.

Tell Claude your situation. It diagnoses, recommends plays, and scripts your talking points.

## What's inside

- Situation diagnosis with interrogation questions
- Exec archetypes (Scope Creeper, Brilliant Jerk, Chaos Agent, Invisible Decision Maker...)
- 10 plays with exact language you can use in the room
- Counter-move preparation
- Combined play sequences for complex scenarios
- Bidirectional lens for when you ARE the exec
- Routing matrix by career stage (Jr PM → CPO)

Free. No gate.
